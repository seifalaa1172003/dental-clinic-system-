# dental-clinic-system-
Dental Clinic Management System – Java desktop app for managing appointments, prescriptions, patient records &amp; staff. Features role-based menus for Doctors, Patients &amp; Receptionists with file-based storage. OOP design with Java Time API.
Doctor Features

    Create Prescriptions – Generate digital prescriptions with medications and instructions for patients

    View Daily Appointments – Display all appointments scheduled for a specific date

    Access Patient Information – View complete patient history including medical records, vitals, and blood type

    Manage Availability – Add, delete, or modify working days and hours

    View Receptionist Contact – Get contact information of receptionist on duty

    Update Schedule – Change all availability at once or edit hours for specific days
    
  Patient Features

    Book Appointments – Reserve available 30-minute time slots with preferred doctor

    Cancel Appointments – Cancel existing booked appointments

    View Available Slots – Display all upcoming available appointments

    Update Personal Information – Change email, mobile number, height, and weight

    Search for Doctors – Find doctors by name or mobile number

    View Service Prices – Check pricing for all dental services (examination, cleaning, fillings, root canal, implants, etc.)

    Medical History Tracking – Automatic recording of all prescriptions in patient history
    
  Receptionist Features

    Reserve Appointments for Patients – Book appointments on behalf of patients

    Cancel Patient Reservations – Cancel existing bookings

    Update Contact Details – Change email and mobile number
    
  System Features
  
     Persistent Data Storage – All data saved to clinic_data.txt file automatically

     Login & Signup System – Secure authentication for all user types

     Role-Based Access Control – Different menus and permissions for each role

     Data Validation – Input validation for emails, Egyptian mobile numbers, blood types, dates, and times

     Automatic Appointment Generation – Creates 30-minute appointment slots based on doctor availability 
     
  Appointment System

    30-Minute Time Slots – Standardized appointment duration

    Status Tracking – Appointments marked as "available" or "booked"

    Date & Time Validation – Prevents invalid bookings

    Doctor Availability Management – Set weekly schedules with custom hours per day
